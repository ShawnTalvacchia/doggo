### Naming Convention

All filenames use kebab-case, saved as `.jpg` in `public/images/generated/`.
Pattern: `{category}-{subject}.jpg`
- Portraits: `{dogname}-portrait.jpg` or `{username}-profile.jpg`
- Posts: `post-{description}.jpg`
- Covers: `community-cover-{location}.jpg`
- Care: `care-{description}.jpg`

### After Generation

Once images are generated, update `mockUsers.ts` to replace Unsplash URLs with local paths:
```
avatarUrl: "/images/generated/tereza-profile.jpg"
```

And update all dog `imageUrl` fields similarly. This can be done in a single pass after all images exist.
