## Tips for Best Results

1. **If an image looks too polished:** Add "taken with smartphone, slight grain, not perfectly composed" to the prompt
2. **If the architecture looks wrong:** Add "Central European architecture, Prague-style facades, art nouveau buildings"
3. **To regenerate with variation:** Click the image in Flow and use "What do you want to change?" to tweak (e.g., "make the dog look more nervous" or "change to overcast lighting")
4. **Pick the best of 2:** Flow generates 2 images per prompt — always check both before deciding
5. **Naming:** Use the suggested filenames so we can map them to mock data easily later
